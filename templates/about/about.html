How is the app build?

We have worked on Colab to write the code for the Sentiment Analysis of a YouTube Video. We have used a pre-trained BERT Model to do sentiment analysis of a YouTuber who is speaking in the video. . We will be using ktrain which is a Python library that makes deep learning more accessible and easier to apply For that we have fetched subtitles of a video from YouTube transcript -API

The sentiments are of the YouTuber’s video, what he/she is it trying to say to its viewers, to know that we have classified the sentiments into 7 emotions(Joy, Sadness, Fear, Anger, Neutral, Love Surprise).

Then we pre-process the text, pass it through BERT Model and plot the learning rate of the model. We have also done an ‘intent based classification’ which will tell us what the intention of a youtuber is on the video. This algorithm works similarly to sentiment analysis.
